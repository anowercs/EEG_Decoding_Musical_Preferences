## ðŸ§  **Experiment Summaries**

### **Experiment 1 â€” Traditional Machine Learning (LOSO)**

* EEG segmented into epochs
* Comprehensive feature extraction (time-domain, frequency-domain, statistical features)
* Classification + regression tasks
* Models include:

  * SVM, Random Forest, KNN, Logistic Regression
  * Ridge, Lasso, RandomForestRegressor
* Evaluation:

  * **Leave-One-Subject-Out (LOSO)**
  * Accuracy, Precision/Recall/F1, RMSE, RÂ²

---

### **Experiment 2 â€” SAOF (Systematic Augmented Operation Framework)**

A novel hybrid framework involving:

* Multi-window EEG slicing
* Feature augmentation
* Noise-enhanced data expansion
* PCA/Dimensionality transformation
* Deep learning models with improved generalization

Achieved:

* **81.8% binary classification accuracy**
* **68.2% cross-subject accuracy**
* Identified important EEG markers (T4, P3, P4, F7, gamma band)
